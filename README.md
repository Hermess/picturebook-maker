# Writing Story

This repository contains the `picturebook-maker` skill package.

`picturebook-maker` turns a story, myth, fable, lesson, or child-facing idea into an interactive picture-book workflow with:

- story-voice selection
- illustrator visual-system selection
- layout rhythm selection
- character-sheet confirmation
- page-by-page generation and review
- stable local text layout
- print-ready packaging

## Files

```text
picturebook-maker/
  SKILL.md
  README.md
  templates/
  references/illustrators/
  scripts/

picturebook-maker-skill.zip
```

## Install

Copy the full folder into your Codex skills directory:

```text
~/.codex/skills/picturebook-maker
```

The zip file contains the same skill package for easy download and transfer.

## Important Rule

The skill is interactive by default. It confirms story direction, visual system, layout rhythm, character design, each generated page, cover, and final contact sheet before producing a print package.

It does not ask models to imitate a living creator's exact style. Named creators are distilled into reusable narrative or visual systems.
